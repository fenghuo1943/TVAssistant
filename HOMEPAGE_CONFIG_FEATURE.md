# 首页快捷方式配置功能说明

## 功能概述

已将首页的 4 项快捷方式（TV 直播、央视片库、抖音、腾讯视频）改为可配置，用户可以在设置页面的"网址管理"中自定义显示哪些快捷方式。

## 修改内容

### 1. 数据结构调整 (`src/settings.ts`)

- 将 `shortcuts` 重命名为 `defaultShortcuts`，作为默认快捷方式配置
- 在 `AppSettings` 类型中添加 `enabledShortcuts: string[]` 字段，用于存储启用的快捷方式 URL
- `defaultSettings` 中默认启用所有快捷方式

### 2. 首页逻辑更新 (`src/HomePage.vue`)

- 使用计算属性 `shortcuts` 动态过滤显示的快捷方式
- 根据 `settings.enabledShortcuts` 过滤默认快捷方式列表

### 3. 设置页面增强 (`src/components/SettingsPanel.vue`)

新增"网址管理"页面，包含以下功能：

#### 界面布局
- 左侧边栏显示"网址管理"菜单项
- 右侧内容区显示所有可用的快捷方式列表
- 每个快捷方式显示：
  - 图标（emoji）
  - 名称
  - 状态（已添加/未添加）
  - 操作按钮（移除/添加）

#### 交互功能
- **键盘导航**：
  - ↑/↓：上下切换快捷方式项
  - →：从侧边栏进入内容区
  - ←：从内容区返回侧边栏
  - Enter：切换快捷方式的启用状态
- **鼠标操作**：
  - 点击"添加"按钮：启用该快捷方式
  - 点击"移除"按钮：禁用该快捷方式

#### 样式设计
- 参考用户提供的图片布局
- 已添加的快捷方式显示绿色"已添加"状态和灰色"移除"按钮
- 未添加的快捷方式显示灰色"未添加"状态和蓝色"添加"按钮
- 支持焦点状态高亮显示

## 使用方法

1. 打开应用，点击首页右上角的齿轮图标进入设置
2. 在左侧边栏选择"网址管理"
3. 使用键盘或鼠标操作：
   - 按 ↑/↓ 键选择要修改的快捷方式
   - 按 Enter 键或点击按钮切换状态
4. 返回首页后，只会显示已启用的快捷方式

## 技术细节

### 快捷键图标映射
```typescript
{
  'TV 直播': '📺',
  '央视片库': '🎬',
  '抖音': '🎵',
  '腾讯视频': '🎞️'
}
```

### 状态管理
- 启用状态存储在 `settings.enabledShortcuts` 数组中
- 数组元素为快捷方式的 URL
- 默认包含所有 4 个快捷方式的 URL

## 注意事项

- 至少需要保留一个快捷方式，否则首页将不显示任何内容
- 修改配置后实时生效，无需重启应用
- 启动模块选择下拉框中的选项会动态反映当前启用的快捷方式
